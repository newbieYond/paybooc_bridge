💡 사용 방법

이 페이지에는 eruda 디버거가 자동으로 로드되어 있습니다. 화면 우측 하단의 eruda 버튼을 클릭하여 개발자 도구를 열어보세요!

🚀 Eruda 기능

  • Console - 콘솔 로그 확인
  • Elements - DOM 요소 검사
  • Network - 네트워크 요청 모니터링
  • Resources - 리소스 및 스토리지 관리
  • Sources - 소스 코드 확인
  • Info - 디바이스 정보
  • Snippets - 코드 스니펫 실행
  • Settings - 설정 관리
